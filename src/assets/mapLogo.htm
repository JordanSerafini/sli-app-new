Obtenez des icônes gratuites dans le style Windows 11 Color

Windows 11 Color est un pack d'icônes plates et éclatantes.

Ces illustrations colorées s'adapteront à toute application de bureau, Web et aux écrans d'applications mobiles. Tout comme les packs d'icônes Cute Color, Hand Drawn Color et Color, les icônes Windows 11 Color peuvent être utilisées comme images clipart pour des infographies, cartes personnalisées ou présentations.

Téléchargez les icônes Windows 11 Color dans des thèmes variés pour vos besoins en matière d'interface utilisateur et de création graphique. Vous pouvez aussi modifier directement ici les images que vous voulez.

Ces illustrations gratuites sont parfaites au pixel près pour s'adapter à votre design et sont disponibles aux formats PNG et vecteur.
